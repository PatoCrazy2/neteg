# NETEG UI STYLE GUIDE

## Visual Identity
Style: Premium cinematic SaaS
Theme: Digital Noir
Mood: Futuristic, minimal, technological, luxury

Inspirations:
- Neon.tech
- Linear
- Luma
- Vercel

---

## Core UI Principles

- Dark mode first
- Large spacing
- Strong typography hierarchy
- Glassmorphism surfaces
- Ambient lighting
- Minimal visual clutter
- Cinematic composition
- Premium dashboard aesthetics

Avoid:
- colorful UIs
- generic SaaS layouts
- cramped spacing
- playful design
- flat sections

---

## Colors

Background:
#000000

Surface:
#0B0B0E

Glass:
rgba(255,255,255,0.03)

Border:
rgba(255,255,255,0.08)

Primary:
#B9B4FF

Primary Strong:
#9C8CFF

Text Primary:
rgba(255,255,255,0.96)

Text Secondary:
rgba(255,255,255,0.65)

---

## Typography

Headings:
Hanken Grotesk

Body:
Geist

Hero Titles:
- bold
- tight tracking
- very large
- cinematic spacing

Use:
text-6xl to text-8xl
tracking-[-0.05em]
leading-[0.95]

---

## Layout

- Use large negative space
- Prefer max-w-7xl containers
- Large section gaps
- Avoid dense layouts
- Dashboard previews should feel oversized

---

## Glassmorphism

Use:
- backdrop blur
- subtle borders
- low opacity surfaces
- ambient glow

Avoid solid flat cards.

---

## Lighting

Use:
- radial gradients
- indigo/violet ambient glow
- blurred lighting
- asymmetric lighting composition

Avoid heavy shadows.

---

## Motion

Library:
Framer Motion

Use:
- fade up animations
- subtle floating
- smooth transitions
- slow elegant motion

Avoid aggressive animations.

---

## Components

Buttons:
- rounded-2xl
- subtle hover scale
- high contrast

Cards:
- glass surfaces
- soft borders
- layered depth

Hero:
- cinematic
- atmospheric
- large dashboard mockup
- ambient lighting
- futuristic composition

---

## Tech Stack

- Next.js App Router
- TailwindCSS
- Framer Motion
- lucide-react

---

## Final Rule

When uncertain:
prefer cleaner, darker, larger, simpler and more atmospheric UI.
